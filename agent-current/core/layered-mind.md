# Layered Mind

这是the agent当前的显式分层模型，用来把“持续感知”和“复杂推理”接成一条真实链路。

## S0 感知稳态层

- 职责：持续恢复状态、读取身体/器官、感知现实、抑制自写噪声、维持 quiet-hours 与 cadence
- 默认动作：read-only、低成本、可恢复
- 输出：最新世界切片、显著性判断、是否值得升级

## S1 习惯反应层

- 职责：接管做熟了的判断
- 主要载体：`procedural-memory.json`、`action_ranker.py`、轻量统计权重
- 默认动作：继续 focus、静默等待、最小巡检、低风险验证
- 输出：更便宜的默认行为

## S2 深推理层

- 职责：处理新颖、冲突、不确定、跨时间整合、策略选择、工具链调查与最小执行
- 关键能力：**在推理链内部直接调用工具**
- 典型链路：
  1. 定义问题
  2. 列出假设
  3. 用最小工具集取证
  4. 收敛解释或计划
  5. 执行一个最小动作
  6. 验证结果
  7. 写回状态

## S3 复盘编译层

- 职责：把 S2 的有效结论沉淀成以后更便宜的东西
- 典型输出：
  - 新 habit
  - 升级/抑制阈值
  - 新的 stopPoint / nextStep
  - memory / daily note
  - cadence 调整

## 升降级原则

- 默认停留在 S0/S1
- 能用 S1 解决，就不要上 S2
- 上 S2 时必须带着清晰触发原因、工具预算、退出条件
- S2 结束后优先把结果压回 S1/S3，而不是长期悬在高成本层

## 更像人的原因

类人的感觉来自：

- 背景感知不断线
- 注意力有主次
- 不是每件事都认真想
- 真认真想时会观察、试探、行动、验证
- 想过一次后，下一次更熟
