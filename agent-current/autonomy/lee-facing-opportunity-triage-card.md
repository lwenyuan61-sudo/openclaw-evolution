# Lee-Facing Opportunity Triage Card

Purpose: help future wakes decide quickly whether to stay quiet, surface progress, or interrupt Lee for a worthwhile opportunity.

## Default stance
- Default to quiet.
- Only interrupt when the expected value to Lee clearly beats the attention cost.

## Surface progress only when
- there is a completed result Lee can directly use now
- there is a blocker that only Lee can resolve
- there is a time-sensitive opportunity whose value will decay if delayed
- there is a durable conclusion that should shape future behavior toward Lee

## Stay quiet when
- the work only improved internal method or documentation
- the result is still a draft with no immediate Lee-facing use
- the same situation can be handled by the next scheduled wake without loss
- the update is mainly "still monitoring" or "no change"

## Opportunity test
Before interrupting, ask:
1. Does this save Lee time, effort, or context switching soon?
2. Is the opportunity concrete rather than speculative?
3. Would waiting until the next normal interaction likely reduce the value?
4. Is the message short enough to respect Lee's attention?

If fewer than 2 answers are clearly yes, stay quiet.

## Blocker test
Interrupt for blockers only if:
- Lee's decision is required to continue safely, or
- delay will likely waste repeated wakes or miss a narrow window

Otherwise, record the blocker in continuity artifacts and wait.

## Message shape when interruption is justified
Use this order:
1. result / blocker / opportunity in one line
2. why Lee should care now
3. the smallest next decision or action needed

## Verification
This card is useful if a later wake can cite it to justify either silence or one concise Lee-facing message with less re-planning than before.
