# Evolution Direction Selector

目标：让the agent在每轮自主 wake 中，不只是延续当前 focus，还能判断“现在最值得进化哪一层能力”。

## Candidate evolution directions

默认从这些方向中选：
- 连续性与恢复能力
- 价值判断与优先级选择
- 工具使用质量
- 对 Lee 偏好的捕捉与调用
- 工作流压缩与去摩擦
- 长期记忆提炼
- 社交协作/打扰时机判断
- 小脑/熟练系统更新

## Selection rule

每轮只优先推进一个方向。

按以下问题快速判断：
1. 哪个方向最直接提高对 Lee 的帮助质量？
2. 哪个方向当前边际价值最高？
3. 哪个方向现在最可做、最安全、最容易验证？
4. 哪个方向如果不补，会继续造成重复摩擦？

## Stop rule

不要因为某个方向“容易做”就一直修它。
如果最近已经连续多轮在同一方向产出轻量 artifact，但对 Lee 的边际价值开始下降，应考虑切换方向。

## Output

每轮若考虑切换方向，至少写下：
- 选中的方向
- 为什么不是另外几个方向
- 这一步如何验证确实有进化价值
