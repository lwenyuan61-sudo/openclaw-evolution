# Scoring

对每个候选动作，快速给 1-5 分：

- Value：对 Lee 的直接或长期价值
- Feasibility：现在能否独立完成
- Safety：低风险、可逆程度
- Continuity：是否延续当前路线
- Non-annoyance：是否无需打扰或只在有成果时打扰

## Decision rule

- 默认选择总分最高的候选动作
- 如果两个动作接近，优先：
  - 更可验证的
  - 更少依赖 Lee 额外输入的
  - 更能改善长期能力的

## Anti-busywork rule

若 Value < 3，则除非它能明显提升未来效率，否则不做。
若 Feasibility < 3，则不要硬做，转为 blocker 或降级方案。
