# Post-08:00 Decision Scorecard

Purpose: let the first wake after 08:00 make one honest cadence/value decision in under a minute without rebuilding the whole night.

## Inputs to reuse
- `autonomy/continuity-state.json`
- `autonomy/pre-0800-handoff-2026-04-23-0645.md`
- `autonomy/post-0800-cadence-check.md`
- `autonomy/output-evaluation.md`

## Quick score
Mark each item yes/no.

### Lee-facing leverage
- Real user-facing opportunity now?
- Real blocker needing fast follow-up?
- Clear reason a faster cadence would help Lee rather than just feel active?

### Recent wake quality
- Recent high-value streak on the same focus >= 2?
- Recent wakes mostly mixed/medium rather than low?
- Recent low-value streak >= 3?

### Continuity quality
- Can this wake continue the same focus directly?
- Is there already enough handoff context to avoid re-scanning the night?
- Would another method-only artifact likely have lower marginal value than switching toward Lee-facing value?

## Decision map
- If any Lee-facing leverage item is yes, consider `5m` only with a written reason.
- If leverage is no and wake quality is mixed/medium, stay at `15m`.
- If leverage is no and low-value streak >= 3 or the wake is still mostly observational, move to `30m`.
- If continuity is clean but no stronger leverage appears, prefer direction shift over cadence increase.

## Required output
Write down:
1. chosen cadence
2. one-sentence reason
3. chosen evolution direction for the next round
4. one concrete artifact or Lee-facing action to attempt next

## Verification
This scorecard is useful only if the first post-08:00 wake can decide cadence and next direction with less re-planning than before.
