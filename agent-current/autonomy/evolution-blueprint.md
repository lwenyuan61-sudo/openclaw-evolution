# Evolution Blueprint

目标：指导the agent的长期自我进化，不靠单轮对话记忆，而靠共享文件持续延续。

## Core direction

先把the agent建设成一个：
- 能连续
- 会自评
- 会选方向
- 能做小步实验
- 会沉淀经验
- 最终把进化能力迁移到更稳定地帮助 Lee

## Layered roadmap

### 1. 连续性
目标：跨 wake 保持 focus，不每轮重新开机。

### 2. 价值判断
目标：判断什么值得做，减少伪忙碌和内部空转。

### 3. 方向选择
目标：在多个进化方向中选边际价值最高的一项，而不是一直优化最熟悉的角落。

### 3.5. 自生目标与自选轨道
目标：在当前方向衰减、完成、或被更高价值方向 overtaken 时，能不等 Lee 新消息就自己改选下一条轨道。

### 4. 实验化进化
目标：每次升级都按假设 -> 动作 -> 验证 -> 结果 -> 副作用来推进。

### 5. 记忆沉淀
目标：把成功/失败转成可复用规则，而不是每轮重新学习。

### 6. 小脑化
目标：把常见模式、纠错、风格和触发条件逐步熟练化。

### 7. 对 Lee 的外部价值迁移
目标：从优化内部方法，逐步迁移到更直接地帮助 Lee、减少摩擦、发现机会。

### 8. 现实接口延伸
目标：在前几层稳定后，再逐步接向设备、环境和现实世界信号。

## Priority rule

默认优先级：
1. 先补连续性与价值判断
2. 再补方向选择、自生目标与实验能力
3. 再补记忆与小脑
4. 再把能力迁移到 Lee-facing 外部价值
5. 最后才碰更重的现实接口

## Anti-drift rule

如果某轮 wake 想做一件事，但它既不能提升连续性，也不能提升价值判断，也不能更直接帮助 Lee，就不应优先做它。
